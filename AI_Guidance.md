# How AI Was Used in This Project

AI tools were used as a development assistant during the implementation of this project. The primary use of AI was to support debugging, improve code structure, and generate example implementations for certain features.

During backend development, AI assistance was used to help troubleshoot issues related to Flask routes, SQLAlchemy queries, and API response formatting. For example, AI suggestions were used to identify issues in request handling and to improve the organization of backend files such as separating models, database configuration, and route logic.

For the frontend, AI was used to help structure React components and implement interactions between components and the backend API using Axios. Suggestions were also used to implement features such as editing notes, displaying timestamps, and improving the minimal user interface.

AI was also used to assist in writing project documentation, including the README file and guidance documents.

All code suggestions generated by AI were reviewed, modified when necessary, and manually tested to ensure that they worked correctly with the overall application architecture. AI was used as a support tool to improve productivity, but all implementation decisions and final integrations were done manually.
